##  安装
### 1.手动添加:<br>
*   1.将 JHDropDownView 文件夹添加到工程目录中<br>
*   2.导入 JHDropDownView.h

### 2.CocoaPods:<br>
*   1.在 Podfile 中添加 pod 'JHDropDownView'<br>
*   2.执行 pod install 或 pod update<br>
*   3.导入 JHDropDownView.h
